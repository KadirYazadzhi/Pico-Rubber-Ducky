# Create a Hidden Folder

Creates a hidden folder in a specific location using the command line.

```
DELAY 500
GUI r
DELAY 200
STRING cmd /c mkdir C:\\Users\\User\\Documents\\HiddenFolder && attrib +h C:\\Users\\User\\Documents\\HiddenFolder
ENTER
```
