# Clearing Browser History

Clears the browser history on Windows using a shell command.

```
DELAY 500
GUI r
DELAY 200
STRING shell:::{3C6E4A2F-1E88-4B60-A6B5-6B905F01A25A}
ENTER
DELAY 500
STRING "history::clear"
ENTER
```
